# MediRutas
Nombre del proyecto: MediRutas

Autores:
Juan Esteban Gonzalez Avila
Tomas Montoya Suarez
Lina Alvarez
Andres Felipe Lopez

Descripción:
Una aplicación que ayude con la logistica de una empresa de transporte para medicos y otros servicios.

Instrucciones:

Administradores:

1 Generar codigos de accesos para cada rol en la empreza

2 Crear y guardar usuarios y contraseñas, distinguiendo de: Jefe, Administradores, Conductores

2 Crear cada tipo de servicios disponibles para los administradores:
-Crear rutas
-Cuentas de cobro
-Reportar problemas
-Ver documentación de las rutas completas
-Ver la lista de usuarios en la aplicación

3 Permitir que el Jefe reparta los tipos de servicios disponibles a los Administradores

Conductores:

1 Crear la forma de ver las rutas asignadas

2 Crear un formulario para realizar en cada ruta al inicio y fin de la misma

3 Permitir ver la cuenta de cobro sobre los servicios realizados

Adiciónal:
- Crear un mapa con acceso a las ubicaciones de los conductores para los administradores
